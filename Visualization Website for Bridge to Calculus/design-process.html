Design Choices

We ideated beyond line charts to test our limits of creativity, however we decided line charts were the easiest to consume especially since we were dealing with time-series data. We wanted to respect the principle of using line charts for time-related data.

We felt that only using line charts would be boring and also ineffective. Adding variation piques curiosity so we wanted to implement a bar chart. Bar charts are great for comparing a relatively small amount of quantitative data for categorical variables, so it presented itself as an ideal encoding mechanism. So, we opted to employ a bar chart to compare demographic scores for a particular year. Out of this grew our idea to implement a slider which would produce different bar charts depending on the year of choice.
